一、毛泽东思想

1. 形成发展历程

时期 历史阶段 理论成果 特点
萌芽时期 国民革命时期（1924-1927） 新民主主义革命思想开始萌芽 结合中国国情探索革命道路
初步形成 土地革命战争时期（1927-1937） 《中国的红色政权为什么能够存在》《星星之火，可以燎原》等 提出农村包围城市、武装夺取政权
成熟时期 抗日战争时期（1937-1945） 《实践论》《矛盾论》《新民主主义论》 思想体系基本成熟
继续发展 解放战争时期及建国初期 新民主主义革命理论系统完成，提出人民民主专政理论 指导中国革命取得胜利

2. 毛泽东思想的科学体系

主要内容包括：

  • 新民主主义革命理论（总路线、革命对象、动力、领导、前途）
  • 社会主义革命和建设理论
  • 革命军队建设和军事战略理论
  • 政策与策略理论
  • 思想政治工作和文化工作理论
  • 党的建设理论

3. 毛泽东思想的地位

  • 马克思列宁主义在中国的运用和发展
  • 被实践证明的正确理论原则和经验总结
  • 中国共产党集体智慧的结晶
  • 党和国家长期指导思想

4. 毛泽东思想的活的灵魂（三大基本方面）

  1. 实事求是：从实际出发，理论联系实际
  2. 群众路线：一切为了群众，一切依靠群众，从群众中来，到群众中去
  3. 独立自主：坚持走自己的路，不照搬照抄外国经验

5. 历史地位

  • 中国革命和建设的科学指南
  • 中国化马克思主义第一次历史性飞跃的理论成果
  • 推动了中国革命从失败走向胜利

二、 新民主主义革命理论

一、近代中国的社会性质与主要矛盾

  • 社会性质：半殖民地半封建社会（帝国主义入侵 + 封建主义腐朽）。
  • 根本原因：帝国主义的侵略。
  • 主要矛盾：帝国主义与中华民族的矛盾、封建主义与人民大众的矛盾。
  • 两大历史任务：民族独立与人民解放。

二、革命性质与革命对象

  • 革命性质：资产阶级民主革命的新形式，即无产阶级领导的民主革命。

  • 革命对象：

    • 帝国主义
    • 封建主义
    • 官僚资本主义（旧民主主义中没有这项，是新民主主义的新增对象）

  • 革命动力：

    • 工人阶级（领导力量）
    • 农民阶级（主力军）
    • 城市小资产阶级（广泛基础）
    • 民族资产阶级（有两面性，可联合可斗争）

三、新旧民主主义革命的区别

比较项 旧民主主义革命 新民主主义革命
指导思想 资产阶级民主主义 马克思列宁主义
领导阶级 资产阶级 无产阶级
革命对象 封建主义 帝国主义、封建主义、官僚资本主义
前途 资本主义 社会主义

根本标志：无产阶级领导。

四、革命道路与基本经验

  • 道路：农村包围城市、武装夺取政权。

  • 提出与理论形成：

    • 1930年《星星之火，可以燎原》：提出农村包围城市思想。
    • 《中国的红色政权为什么能够存在》《井冈山的斗争》：阐述根据地建设。

  • 主要斗争形式：武装斗争。

  • 三大法宝：统一战线、武装斗争、党的建设。

五、新民主主义革命总路线

  • 毛泽东于1939年在《〈中国革命和中国共产党〉》中首次提出。

  • **1948年《在晋绥干部会议上的讲话》**中完整表述：

    无产阶级领导的人民大众的、反帝反封建反官僚资本主义的新民主主义革命。

  • 核心：领导权问题——无产阶级必须领导革命。

六、新民主主义政治、经济、文化纲领

政治纲领
建立无产阶级领导的各革命阶级联合专政的新民主主义共和国。
不是苏维埃制，而是联合专政。

经济纲领

  • 没收封建地主土地归农民所有。
  • 没收官僚资本归国家所有。
  • 保护民族工商业。

（这三条构成新民主主义经济的“三大支柱”）

文化纲领
新民主主义文化是：

  • 无产阶级领导的；
  • 民族的；
  • 科学的；
  • 大众的。

七、重要文献与会议

  • 五四运动（1919）：旧民主主义向新民主主义革命转折点。
  • 中共二大（1922）：提出彻底反帝反封建的民主革命纲领。
  • 八七会议（1927）：“枪杆子里面出政权”。
  • 遵义会议（1935）：确立毛泽东在中共的领导地位。
  • 1939《〈中国革命和中国共产党〉》：首次提出新民主主义革命概念。
  • 1940《新民主主义论》：系统阐述新民主主义革命理论。
  • 1948《在晋绥干部会议上的讲话》：完整表述总路线。

八、统一战线的策略

  • 对象与策略：

    • 对顽固派：有理、有利、有节。
    • 对中间力量（民族资产阶级）：又联合又斗争。

  • 统一战线历程：

    1. 第一次国共合作（国民革命）
    2. 工农民主统一战线（土地革命）
    3. 抗日民族统一战线
    4. 人民民主统一战线（解放战争后）

九、革命成功的原因与经验

  • 根本原因：坚持无产阶级领导。
  • 三大法宝：统一战线、武装斗争、党的建设。
  • 关键问题：农民问题——中国革命的中心问题。

三、 社会主义改造理论

一、过渡时期的时间与性质

  • 时间范围：1949年新中国成立到1956年社会主义改造基本完成。
  • 社会性质：新民主主义社会（属于社会主义范畴）。
  • 任务：实现国家社会主义工业化，并逐步完成对农业、手工业、资本主义工商业的社会主义改造。

二、过渡时期总路线

  • 内容：

    “一化三改”——即以社会主义工业化为主体，同时逐步实现对农业、手工业、资本主义工商业的社会主义改造。

  • 特点：

    • 工业化与改造同时并举；
    • 建设与革命同时进行；
    • 改造经济与改造人相结合。

三、三大改造

  1. 农业的社会主义改造

    • 方式：互助组 → 初级社 → 高级社 → 人民公社。
    • 性质：从个体经济到集体经济。
    • 政策：自愿互利、典型示范、国家帮助。
    • 1956年基本完成。

  2. 手工业的社会主义改造

    • 方式：手工业供销合作社 → 生产合作社。
    • 特点：以合作化为主要途径。

  3. 资本主义工商业的社会主义改造

    • 方式：国家资本主义形式，如“公私合营”。
    • 政策：和平赎买。
    • 结果：形成全民所有制（国营）与集体所有制两种经济成分。

四、社会主义制度的确立

  • 1956年三大改造完成：标志着社会主义基本制度确立。
  • 国家政权性质：无产阶级专政。
  • 经济结构：公有制为主体，多种经济成分并存。
  • 政治生活主题：正确处理人民内部矛盾。

五、重要会议与方针

  • 七届二中全会（1949）：提出“两个务必”——谦虚谨慎、不骄不躁；艰苦奋斗。
  • 中共八大（1956）：提出主要矛盾转化为人民日益增长的物质文化需要与落后的社会生产之间的矛盾。

六、历史意义

  • 标志着从新民主主义向社会主义的历史性飞跃；
  • 社会主义基本制度确立，中国进入社会主义初级阶段；
  • 为之后的社会主义建设奠定制度基础。

七、相关概念速记

概念 含义
一化三改 工业化 + 农业、手工业、资本主义工商业改造
和平赎买 对民族资产阶级采取和平方式转为社会主义经济
公私合营 国家与私人资本合作经营的过渡形式
新民主主义社会 向社会主义过渡的社会形态
两个务必 谦虚谨慎、不骄不躁；艰苦奋斗

📚 总结脉络

新民主主义革命 → 新民主主义社会 → 社会主义改造 → 社会主义制度确立
这是中国现代史从“民族独立”到“制度建设”的完整链条。革命解决“谁掌权”，改造解决“怎么建国”。

四、社会主义建设道路初步探索的理论成果（1956—1976）

一、历史背景

  1. 时间起点：1956年社会主义改造基本完成，中国进入全面建设社会主义的新时期。
  2. 主要任务：探索什么是社会主义、怎样建设社会主义。
  3. 客观困难：缺乏建设社会主义的经验，只能“摸着石头过河”。

二、主要理论成果（可概括为“五个探索”）

（1）社会主义社会的主要矛盾理论

  • 提出者：毛泽东。
  • 内容：社会主义社会的基本矛盾是人民日益增长的物质文化需要同落后的社会生产之间的矛盾。
  • 意义：指出了社会主义社会仍然存在矛盾，否定了“阶级斗争为纲”的极端观点。
  • 后来在1978年被邓小平重新强调，成为改革开放的理论起点。

（2）关于社会主义建设总路线

  • 时间：1956年提出、1958年正式表述。
  • 内容：“鼓足干劲，力争上游，多快好省地建设社会主义。”
  • 特点：强调速度和积极性，但忽视了经济规律，导致“大跃进”的错误。
  • 启示：建设社会主义必须尊重经济规律、实事求是。

（3）关于经济建设方针的探索

  • 提出：重工业优先、农业基础、轻重并举。
  • 成果：初步建立起比较完整的工业化基础体系。
  • 问题：过分强调重工业，忽视农业和轻工业的发展，经济比例失衡。

（4）关于政治建设和民主法制

  • 成果：提出“人民代表大会制度”“中国共产党领导的多党合作和政治协商制度”。
  • 问题：在实践中过于强调政治运动，忽视法制建设。
  • 教训：政治建设要制度化、法治化。

（5）关于文化建设

  • 成果：强调教育为无产阶级政治服务，重视思想改造。
  • 问题：文化大革命时期走向极端，否定知识分子作用。
  • 启示：社会主义文化建设必须坚持百花齐放、百家争鸣。

三、意义与教训

意义：

  • 为中国探索社会主义建设道路积累了宝贵经验。
  • 提出了社会主义社会发展规律的一些初步认识。
  • 为后来的改革开放奠定思想基础。

教训：

  • 对社会主义建设规律认识不够，主观主义、急躁冒进。
  • 没有很好地处理改革、发展、稳定的关系。

总评价一句话：
——“成就辉煌、探索艰辛、经验宝贵、教训深刻。”

五、中国特色社会主义理论体系的形成与发展（1978—至今）

一、形成的历史背景

  1. 1978年党的十一届三中全会：以邓小平为核心的领导集体重新确立解放思想、实事求是的思想路线。
  2. 任务转变：从“以阶级斗争为纲”转向“以经济建设为中心”。
  3. 开启了中国特色社会主义新时期。

二、理论体系的主要内容与发展脉络

（1）邓小平理论——中国特色社会主义的开创

  • 时间：形成于十一届三中全会到十五大（1978—1997）。
  • 核心内容：
    • 什么是社会主义、怎样建设社会主义。
    • 坚持社会主义初级阶段理论。
    • 基本路线：“一个中心，两个基本点”。
    • 改革开放、发展生产力、共同富裕。
    • “发展才是硬道理”。
  • 意义：科学回答了建设中国特色社会主义的总任务和基本路线。

（2）“三个代表”重要思想——中国特色社会主义的继续发展

  • 提出者：江泽民，形成于1997—2002。
  • 核心内容：
    • 中国共产党要始终代表中国先进社会生产力的发展要求，
    • 代表中国先进文化的前进方向，
    • 代表中国最广大人民的根本利益。
  • 意义：回答了“建设什么样的党、怎样建设党”的问题。

（3）科学发展观——中国特色社会主义的进一步深化

  • 提出者：胡锦涛，形成于2003—2012。
  • 核心内容：
    • 以人为本、全面协调可持续发展。
    • 建设社会主义和谐社会。
  • 意义：在新的发展阶段完善了社会主义现代化建设的战略思想。

（4）习近平新时代中国特色社会主义思想——进入新时代的理论升华

  • 时间：十八大以来（2012—至今）。
  • 核心内容：
    • 坚持和发展中国特色社会主义。
    • 总任务：实现中华民族伟大复兴。
    • “五位一体”总体布局，“四个全面”战略布局。
    • 新发展理念：创新、协调、绿色、开放、共享。
    • 推进国家治理体系和治理能力现代化。
  • 意义：马克思主义中国化时代化的最新成果。

三、理论体系的逻辑关系

你可以把它看作一条时间轴上的理论“接力棒”：

毛泽东思想 → 奠定基础
邓小平理论 → 开创新局
三个代表 → 发展党的建设理论
科学发展观 → 强调可持续与以人为本
习近平新时代思想 → 推进国家全面现代化

四、总体意义

  • 形成了一个不断发展、与时俱进的科学体系。
  • 统一于建设中国特色社会主义的伟大实践。
  • 是当代中国马克思主义、21世纪马克思主义。

如果你想更方便记忆，可以抓住这两章的关键词：

  • 初步探索：五个方面探索 + 成就与教训
  • 理论体系：邓小平起、习近平成；发展脉络、时代任务、核心理念

六、邓小平理论

1. 形成背景

  • “文化大革命”结束后，国家进入改革开放新时期
  • 面临“什么是社会主义、怎样建设社会主义”的根本问题

2. 理论形成过程

  1. 思想解放（1978年中央工作会议提出“解放思想，实事求是”）
  2. 理论体系形成（十一届三中全会后确立改革开放路线）
  3. 理论体系完善（1992年南方谈话，十四大正式提出“邓小平理论”）

3. 核心内容

  • 本质问题：什么是社会主义，怎样建设社会主义
  • 基本理论：社会主义初级阶段
  • 基本路线：一个中心、两个基本点
  • 发展观：发展是硬道理
  • 精髓：解放思想，实事求是
  • 目的：实现共同富裕
  • 政治保证：坚持四项基本原则
  • 动力：改革开放

4. 理论地位

  • 马克思主义与中国实际结合的第二次飞跃
  • 中国特色社会主义理论体系的开端
  • 被称为当代中国的马克思主义

七、“三个代表”重要思想（江泽民）

1. 内容

党必须始终代表：

  1. 中国先进社会生产力的发展要求；
  2. 中国先进文化的前进方向；
  3. 中国最广大人民的根本利益。

2. 核心要义

  • 党的建设的关键：坚持执政为民
  • 理论品格：与时俱进
  • 指导思想地位：十六大写入党章，成为全党指导思想

3. 理论意义

  • 继承和发展了马克思主义、毛泽东思想、邓小平理论
  • 推动中国特色社会主义进入新阶段

八、科学发展观（胡锦涛）

1. 提出背景

21世纪初，中国发展进入关键时期，出现经济社会不协调等问题。

2. 科学内涵

  • 第一要义：发展
  • 核心：以人为本
  • 基本要求：全面协调可持续发展
  • 根本方法：统筹兼顾

3. 目标

实现经济、政治、文化、社会建设全面协调发展。

4. 理论地位

  • 中国特色社会主义理论体系的重要组成部分
  • 马克思主义中国化的最新成果

五、马克思主义中国化的过程与意义

1. 含义

把马克思主义基本原理同中国具体实际相结合，形成具有中国特色的理论体系。

2. 两次历史性飞跃

  1. 毛泽东思想（第一次飞跃）
  2. 邓小平理论及后续思想（第二次飞跃）

3. 原因

  • 马克思主义理论的内在要求
  • 与中国优秀文化的相容性
  • 中国革命、建设、改革的实践需要

六、思想路线：实事求是

1. 提出与发展

  • 最早提出：毛泽东（1929年）
  • 系统论证：《实践论》《矛盾论》《改造我们的学习》
  • 邓小平重新确立实事求是思想路线

2. 含义

从实际出发，理论联系实际，在实践中检验真理和发展真理。

3. 本质

辩证唯物主义和历史唯物主义的统一。
